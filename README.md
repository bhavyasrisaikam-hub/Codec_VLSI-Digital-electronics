### VLSI Design of FIR Filter for DSP Applications

Designed and implemented an FIR filter using Verilog HDL for DSP applications. Filter coefficients were generated using GNU Octave/MATLAB and integrated into the hardware design. The design was simulated, synthesized, and analyzed in Vivado for area, timing, and power performance, demonstrating efficient VLSI implementation.

### FPGA Traffic Light Controller

Designed and implemented a traffic light controller on FPGA using Verilog HDL and Finite State Machine (FSM) concepts. The system controls traffic signals through predefined states and timing sequences. Functional simulation and synthesis were performed in Vivado, validating correct operation and FPGA-based digital control design.
